Métodos de Ordenamiento

_______________________

En computación un algoritmo de ordenamiento es un algoritmo que pone elementos de una lista o un vector en una secuencia ordenada.Los métodos de ordenamiento son importantes ya que con ordenamientos eficientes se optimiza el uso de otros algoritmos (como los de búsqueda y fusión) que requieren listas ordenadas para una ejecución rápida.
